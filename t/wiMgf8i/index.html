Инструкция по подключению ТВ

1. Установите и откройте Happ на телевизоре, затем выберите «Поделиться через Веб».

2. На экране появится 5-значный код (UID).

3. Введите этот код в поле ниже и нажмите "Отправить подписку".

Отправить подписку

Telegram Proxy

Операторы замедляют Telegram. Наш прокси вернёт мессенджеру скорость — подключается в один тап, без VPN.

Добавить устройство

Для добавления дополнительного устройства необходим тариф на несколько устройств. Чтобы увеличить лимит — в главном меню нажмите Продлить, выберите способ оплаты и Показать все лимиты.

Если тариф уже позволяет несколько устройств — запросите добавление через кнопку ниже.

Раздача подписки третьим лицам или её перепродажа запрещены и влекут блокировку аккаунта.
